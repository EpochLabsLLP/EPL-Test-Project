# GovernanceTestApp

Test project for validating the EPL Project Template governance system.

This project exists to exercise and verify the governance framework:
- Hook enforcement (code-gate, commit-gate, dep-gate)
- Session heartbeat (auto trace-check)
- Spec-driven development workflow (PVD → ES → BP → TP → WO)
- Quality gates and module completion
- Template sync from upstream
